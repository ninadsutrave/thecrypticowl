# 🦉 The Cryptic Owl

**Master the art of Cryptic Crosswords, one day at a time.**

The Cryptic Owl is a daily crossword learning app designed to demystify the complex world of cryptic clues. By combining a gamified experience with a structured learning curriculum, it turns a daunting mental exercise into a rewarding daily habit.

---

## ✨ Features

- **🧩 Daily Cryptic Clue**: A hand-crafted, high-quality cryptic clue every single day.
- **💡 Progressive Hint System**: Stuck? Reveal hints step-by-step—from a subtle nudge to the full breakdown of the clue's mechanics.
- **🎓 Interactive Learning Hub**: A comprehensive guide covering the "Big 5" of cryptics: Anagrams, Reversals, Containers, Hidden Words, and more.
- **🔥 Gamified Progress**: Track your daily streaks, earn XP, and level up as you become a cryptic pro.
- **🦉 Meet Ollie**: Your personal guide, Ollie the Owl, who reacts to your progress and cheers you on.
- **📊 Solve History**: Authenticated users can track their entire solve history and see their growth over time.
- **🌙 Beautiful Dark Mode**: A custom-designed interface that's easy on the eyes, day or night.
- **📶 Offline-First**: Solve anywhere. Your progress is saved locally and synced to the cloud when you're back online.

---

## 🚀 How to Onboard

1. **Start with the Daily Puzzle**: Head straight to the [Today's Puzzle](https://thecrypticowl.com/puzzle) and try your hand at the current clue.
2. **Use the Hints**: If it's your first time, don't be afraid to use the hints! They are designed to teach you the *why* behind the answer, not just the *what*.
3. **Visit the Learn Section**: Take 5 minutes to read the [Learn Hub](https://thecrypticowl.com/learn). It breaks down the secret language of cryptic indicators.
4. **Sign In to Sync**: Use your Google account to ensure your streaks and history are never lost, even if you switch devices.
5. **Make it a Habit**: Come back tomorrow for a new challenge!

## 🛠 Tech Stack

- **Frontend**: React 18, TypeScript, Vite
- **Animations**: Framer Motion (motion/react)
- **Styling**: Tailwind CSS v4 + Theme-based Design System
- **Backend**: Supabase (PostgreSQL + Auth)
- **Icons**: Lucide React
- **Components**: Radix UI primitives

---

## 💻 Development

### Prerequisites
- Node.js (v18+)
- A Supabase project (for full features)

### Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/ninadsutrave/thecrypticowl.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set up environment variables:
   Create a `.env` file based on `.env.example` and add your Supabase credentials.
4. Run the development server:
   ```bash
   npm run dev
   ```
